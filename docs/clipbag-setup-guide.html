Étape 1 sur 8
Prérequis — avant tout
Checklist obligatoire avant de dépenser 1€
1
Site en ligne — clipbag.fr accessible, page produit fonctionnelle, panier et checkout Stripe qui marchent.
2
Commande test passée — commande toi-même le produit en mode test Stripe. Vérifie la page de confirmation. Si ça bug, tu le sauras avant tes clients.
3
Compte Facebook Business — va sur business.facebook.com. Crée un Business Manager si tu n'en as pas. Gratuit.
4
Page Facebook ClipBag — Meta exige une page pour diffuser des pubs Instagram. Crée-en une même vide, elle n'est pas visible dans les ads.
5
Carte bancaire prête — nécessaire pour activer le compte pub. Meta débite à chaque seuil (25€ → 50€ → 250€) ou en fin de mois.
Sans ces 5 points validés, chaque euro dépensé en pub est perdu. Ne skip pas cette étape.
Étape 2 sur 8
Créer le compte publicitaire
Dans Business Manager
1
Va sur business.facebook.com → menu hamburger → Comptes publicitaires → Ajouter → Créer un compte pub.
2
Nom du compte : ClipBag — France. Fuseau horaire : Paris. Devise : EUR.
3
Ajoute ta carte bancaire dans Paramètres de paiement. Meta ne débite pas immédiatement.
4
Lie ta Page Facebook ClipBag à ce compte via les actifs Business Manager.
Accès Ads Manager
Une fois le compte créé, accède à Meta Ads Manager via adsmanager.facebook.com. C'est là que tout se passe : campagnes, adsets, créatifs, résultats en temps réel.
Étape 3 sur 8
Installer le Pixel Meta
Créer le pixel dans Ads Manager
1
Dans Ads Manager → Événements Manager → Se connecter aux sources de données → Web → Pixel Meta.
2
Nom : Pixel ClipBag. Récupère ton Pixel ID (16 chiffres).
Intégration dans ton app Vue.js — index.html
<!-- Colle dans le <head> de index.html -->

  ...}

  fbq('init', 'TON_PIXEL_ID');
  fbq('track', 'PageView');

Le code complet est dans Événements Manager → Configurer manuellement → Copier le code.
4 événements obligatoires à déclencher
1
ViewContent — sur ta page produit au chargement.
fbq('track', 'ViewContent')
2
AddToCart — au clic sur "Ajouter au panier".
fbq('track', 'AddToCart')
3
InitiateCheckout — à l'entrée sur la page checkout.
fbq('track', 'InitiateCheckout')
4
Purchase — sur la page de confirmation après paiement Stripe.
fbq('track', 'Purchase', {value: 29.99, currency: 'EUR'})
Installe l'extension Chrome "Meta Pixel Helper" pour vérifier que chaque event se déclenche correctement sur ton site.
Étape 4 sur 8
Créer la campagne TEST
Dans Ads Manager — créer la campagne
1
Clique + Créer. Objectif : Ventes — pas Trafic, pas Notoriété. Ventes uniquement.
2
Nom de la campagne : [TEST] Prospection ClipBag.
3
Active le budget de campagne Advantage+ (CBO) : 7€/jour. Meta répartit automatiquement entre tes adsets.
Paramètres campagne
Type de budgetJournalier CBO
Montant7€ / jour
Stratégie d'enchèreCoût le plus bas automatique

Ne passe pas en "coût par résultat cible" au début. L'algo a besoin de données pour apprendre. Laisse-le en automatique les 2 premières semaines.

Étape 6 sur 8
Uploader les créatifs
Dans chaque adset — répéter 3 fois
1
Dans l'adset → + Ajouter une publicité. Nom : C1 - Problème/Solution.
2
Format : Carrousel pour les créatifs 1 et 3. Image unique pour le créatif 2.
3
Uploade tes photos. Dimensions requises : 1080×1080px (1:1). Poids max : 30MB par image.
4
Colle le texte AIDA dans "Texte principal". Titre : ClipBag — Sac magnétique.
5
URL de destination : clipbag.fr/produit. CTA : Acheter maintenant.
Specs techniques photos
Format1:1 — 1080×1080px
ExtensionJPG ou PNG
Poids max30 MB
Texte sur image< 20% de la surface
Stories bonus9:16 si dispo
Qualité min72 DPI
Meta rejette les images avec trop de texte (règle des 20%). Garde le texte overlay minimal — l'essentiel du message doit être dans le texte d'annonce, pas sur la photo.
Étape 7 sur 8
Campagne Retargeting
Créer une 2e campagne séparée
1
Nouvelle campagne. Objectif : Ventes. Nom : [RETARGETING] Visiteurs ClipBag.
2
Budget : 1€/jour. Pas de CBO nécessaire, 1 seul adset.
Créer l'audience personnalisée
1
Dans l'adset → Audience → Créer une audience personnalisée.
2
Source : Site web (Pixel). Événement : ViewContent — Derniers 14 jours.
3
Exclure : créer une audience Purchase — 30 jours et l'exclure. Tu ne recibles pas ceux qui ont déjà acheté.
Texte d'annonce retargeting
"Tu avais regardé le sac ClipBag. Il reste quelques unités à 29,99€.

⭐⭐⭐⭐⭐ Déjà adopté par des centaines de pratiquants en salle.

Livraison offerte — expédié sous 24h."
Cette campagne sera vide les premiers jours — c'est normal. Le pixel doit collecter des visites avant de pouvoir recibler. Ça démarre vraiment après 3-5 jours de pubs actives.
Étape 8 sur 8
Lancer & surveiller
Jours 1 à 4 — ne touche à rien
L'algo Meta est en phase d'apprentissage. Il teste différents profils, horaires et placements. Toute modification reset l'apprentissage à zéro. Résiste à l'envie de couper.
Métriques à checker — jour 5
MétriqueMauvaisAcceptableBon
CTR lien< 0.8%0.8–1.5%> 1.5%
CPM France> 15€10–15€< 10€
CVR visite→achat< 1%1–2%> 2%
ROAS (7j)< 2x2–3x> 3x
Fréquence> 42–4< 3
Règles de décision — jour 7
1
CTR < 0.8% → Coupe le créatif, change le visuel en priorité.
2
CTR bon mais 0 vente → Problème landing page ou prix, pas la pub.
3
ROAS > 3x sur 7j → Augmente le budget de 20% max (7€ → 8.40€ → 10€…).
4
Fréquence > 4 → Rafraîchis les créatifs, l'audience est épuisée.
Budget total : 7€/j (TEST) + 1€/j (RETARGETING) = 8€/jour = 240€/mois. Dans les clous. Tu es prêt à lancer.
